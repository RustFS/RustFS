# 开发前必读

我们的愿景是：够帮助全人类提升数据安全，降低存储成本。
我们的价值观：简单、诚信。

我们希望更好的人，成为我们的开源伙伴。


# 关于政治、国籍、人种、同性恋、信仰

我们不欢迎团队讨论任何政治倾向、同性念、国籍、信仰和人种等尖锐和敏感的问题。

这是你个人的倾向和个人喜好，我们非常尊重。

但是，我们不希望因此影响开源这一项帮助全人类这件更伟大的事业。

随时欢迎您加入我们的团队。
